# Moj Portfolio

Enostavna, odzivna in čista predloga za osebni portfolio, pripravljena za objavo na platformi Vercel.

## Vsebina

* `index.html` - Glavna struktura strani s predstavitvijo, projekti in kontaktom.
* `style.css` - Stilsko oblikovanje za moderen in minimalističen videz.
* `script.js` - JavaScript datoteka za interaktivnost (npr. prikaz kontaktov).

## Kako uporabiti

1. Prenesite ali kopirajte datoteke v svoj lokalni imenik.
2. Odprite `index.html` in uredite besedilo (ime, opise projektov, kontaktne podatke).
3. Naložite projekt na GitHub in ga povežite z Vercelom za brezplačno objavo na spletu.
